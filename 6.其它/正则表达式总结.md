# 1.Java校验正则

## 1.1.类库

`java.util.regex.Pattern `和` java.util.regex.Matcher`

## 1.2.语法

```java
Pattern pattern = Pattern.compile("");// 编译正则表达式
Matcher match = pattern.matcher("");// 校验字符串
boolean flag = match.matches();//匹配成功返回true，反之返回false
```

# 2.正则表达式总结

## 2.1.身份证

### 2.1.1.十五位身份证

```reStructuredText
^[1-9]\\d{7}((0[1-9])|(1[0-2]))((0[1-9])|([1-2][0-9])|(3[0-1]))\\d{3}$
```

### 2.1.2.十八位身份证

```reStructuredText
^[1-9]\\d{5}[1-9]\\d{3}((0[1-9])|(1[0-2]))((0[1-9])|([1-2][0-9])|(3[0-1]))\\d{3}(\\d|x|X)$
```

### 2.1.3.十五位或十八位

```reStructuredText
^[1-9]\\d{5}[1-9]\\d{3}((0[1-9])|(1[0-2]))((0[1-9])|([1-2][0-9])|(3[0-1]))\\d{3}(\\d|x|X)$|^[1-9]\\d{7}((0[1-9])|(1[0-2]))((0[1-9])|([1-2][0-9])|(3[0-1]))\\d{3}$
```

## 2.2.中文人名

```re
^[\\u4E00-\\u9FA5\\uf900-\\ufa2d·s]{2,20}$
```

## 2.3.特殊符号

```reStructuredText
[`~!@#$%^&*()+=|{}':;',\\\\[\\\\].<>/?~！@#￥%……&*（）——+|{}【】‘；：”“’。，、？]
```

### 2.3.1.去除特殊符号写法

```java
String resource = "%%%6%&^66$$6";// 待校验字符串
String pattern = "[`~!@#$%^&*()+=|{}':;',\\\\[\\\\].<>/?~！@#￥%……&*（）——+|{}【】‘；：”“’。，、？]"; // 匹配特殊字符的正则表达式
Pattern p = Pattern.compile(pattern);// 编译正则表达式
// 将所有特殊字符改为""并将结果去重
String res = p.matcher(resource).replaceAll("").trim();
```

